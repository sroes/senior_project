About

My name is Scott Roes. I'm a CINS major, 25 years old, from Thousand Oaks, have a passion for technology learning new things.
I first created the apps for taks, budget, and journal. I then set up the configurations for them in the settings file. To get taks working I used forms and models to add data to my tasks list. I was able to get data from my model to my views smoothly. For my task pie chart in my dashboard i used chartist.js to display my pie chart. For budget i used similar methods i did from tasks, have forms and models handling data in a sufficient way. Just like my task chart in my dashboard i used chartist.js for my bar chart for budget. My journal was very straight forward, i used data from my views to display the entries and last logged in time in my dashboard for journal. For all apps i had to create seperate html files to support actions such as adding, editing, and deleting. For styling of my dashboard and buttons i used several bootstrap and jQuery libraries.
